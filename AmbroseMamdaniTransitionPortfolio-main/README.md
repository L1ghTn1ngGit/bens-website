# Ambrose A. Sukraj Portfolio

A personal portfolio website showcasing the work, experience, and initiatives of Ambrose A. Sukraj — a NYC-based advocate for public service, policy research, and civic engagement.

## Overview

This portfolio highlights key experiences including:

- **The Marble Project** — A DOE transparency initiative exposing budget misallocation in NYC public schools
- **United Nations Youth Delegate** — Representing NYC youth in international policy discussions
- **NYU Law Institute** — Constitutional law and policy research

## Features

- Responsive design optimized for desktop and mobile devices
- Interactive card carousel with hover-pause functionality
- Smooth animations and transitions
- Video showcase integration
- Accessible navigation with keyboard support

## Local Development

1. Clone the repository:
   ```
   git clone https://github.com/L1ghTn1ngGit/AmbroseMamdaniTransitionPortfolio.git
   ```

2. Open `index.html` in your browser, or use a local server:
   ```
   npx serve .
   ```
